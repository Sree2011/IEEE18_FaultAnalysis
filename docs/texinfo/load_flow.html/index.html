1 Load Flow Analysis ¶

Function File: V, theta, P, Q = load_flow (bus_data, Y, gen_data) ¶

Newton–Raphson load flow solver.

  • Description
  • Inputs
  • Outputs
  • Example

1.1 Description ¶

This function performs Newton–Raphson load flow analysis to compute voltage magnitudes, angles, and power injections at buses.

1.2 Inputs ¶

  • bus_data: Matrix containing bus information
  • Y: Admittance matrix (Ybus)
  • gen_data: Matrix containing generator information

1.3 Outputs ¶

  • V: Voltage magnitudes at buses
  • theta: Voltage angles at buses
  • P: Active power injections at buses
  • Q: Reactive power injections at buses

1.4 Example ¶

% Example usage:
[V, theta, P, Q] = load_flow(bus_data, Ybus, gen_data);
