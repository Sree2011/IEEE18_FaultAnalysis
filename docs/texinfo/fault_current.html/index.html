1 Fault Current (3-Phase) ¶

Function File: If = fault_current_3ph (Vprefault, Z1) ¶

Calculates the three-phase fault current (If) based on prefault voltage (Vprefault) and positive sequence impedance (Z1).

  • Inputs
  • Output

1.1 Inputs ¶

  • Vprefault: Prefault voltage (p.u.)
  • Z1: Positive sequence impedance (p.u.)

1.2 Output ¶

  • If: Three-phase fault current (p.u.)
