1 Fault Current (3-Phase) ¶

Function File: If = fault_current_3ph (Vprefault, Z1) ¶

Calculates the three-phase fault current (If) based on prefault voltage (Vprefault) and positive sequence impedance (Z1).

  • Inputs
  • Output

1.1 Inputs ¶

  • Vprefault: Prefault voltage (p.u.)
  • Z1: Positive sequence impedance (p.u.)

1.2 Output ¶

  • If: Three-phase fault current (p.u.)

2 Fault Current (SLG) ¶

Function File: If = fault_current_SLG (Vprefault, Z0,Z1, Z2) ¶

Calculates the three-phase fault current (If) based on prefault voltage (Vprefault) , positive sequence impedance (Z1), zero sequence impedance (Z0), negative sequence impedance (Z2).

  • Inputs
  • Output

2.1 Inputs ¶

  • Vprefault: Prefault voltage (p.u.)
  • Z0: Zero sequence impedance (p.u.)
  • Z1: Positive sequence impedance (p.u.)
  • Z2: Negative sequence impedance (p.u.)

2.2 Output ¶

  • If: Single line to ground fault current (p.u.)
